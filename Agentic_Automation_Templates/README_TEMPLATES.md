## Overview
This repository contains scalable automation architecture templates for multiple stacks.

### Included Templates
- Node.js + TypeScript
- Python (Package-First)
- .NET 8 Worker
- ArcGIS Pro SDK Add-In

Each template enforces:
- Deterministic execution
- Structured logging
- Continue-on-error policy
- Config-driven runtime
